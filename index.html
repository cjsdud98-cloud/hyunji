    서비스 소개

    현지인 맛집은 광고 순위가 아닌, 지역·거리를 기준으로 동네 맛집을 찾아 주는 가벼운 검색 서비스입니다. 네이버 지역 검색 데이터를 바탕으로 후보를 모아 보여 드립니다.

    📍

    지역 중심 검색

    연남동, 강남역처럼 가고 싶은 동네를 입력하면 그 주변 맛집 후보를 모읍니다.

    📏

    거리 반영 정렬

    「내 위치」로 GPS를 허용하면 내 위치 기준으로, 아니면 검색한 지역 중심으로 가까운 순으로 정렬합니다.

    💬

    가게별 메모·댓글

    목록에서 가게를 누르면 상세 정보와 함께 짧은 후기·팁을 남길 수 있습니다. 이 기기에만 저장됩니다.

    이렇게 사용해 보세요

    1. 검색창에 가고 싶은 동·역·구 이름을 입력합니다.
    2. 필요하면 내 위치를 눌러 GPS를 허용한 뒤, 맛집 검색을 누릅니다.
    3. 목록에서 맛집을 선택해 주소·전화·네이버 링크와 댓글을 확인합니다.

    검색 결과는 네이버 지역 검색 API를 통해 제공되며, 영업 시간·메뉴·가격 등은 각 업체·네이버 페이지에서 최종 확인해 주세요.

    FAQ

    자주 묻는 질문입니다. 더 궁금한 점은 상단 문의를 통해 알려 주세요.

    현지인 맛집은 무료인가요?

    네, 별도 가입 없이 지역 검색·목록 보기·기기 내 댓글 저장 기능을 무료로 이용할 수 있습니다. 데이터 통신 요금은 이용 중인 이동통신사·인터넷 요금제에 따릅니다.

    어떤 기준으로 맛집이 정렬되나요?

    여러 키워드(맛집, 한식, 카페 등)로 네이버 지역 검색을 조회한 뒤 중복을 제거하고, GPS를 쓰면 내 위치에서 가까운 순, GPS가 없으면 입력한 지역 중심에서 가까운 순으로 보여 드립니다. 네이버 API 특성상 완벽한 “현지인만 아는” 순위가 아닐 수 있습니다.

    「내 위치」는 꼭 켜야 하나요?

    아니요. 위치를 허용하지 않아도 지역명만으로 검색할 수 있습니다. 이 경우 거리는 검색한 동네 중심 좌표를 기준으로 계산됩니다.

    댓글은 다른 사람도 볼 수 있나요?

    아니요. 댓글은 브라우저 localStorage에만 저장되며, 같은 기기·같은 브라우저에서만 다시 볼 수 있습니다. 서버에 업로드되지 않습니다.

    목록이 비어 있거나 오류가 나요.

    지역명을 다시 입력해 보세요. 배포 사이트에서는 Cloudflare 환경 변수 NAVER_CLIENT_ID, NAVER_CLIENT_SECRET이 설정되어 있어야 합니다. 로컬에서는 naver-config.json과 node server.mjs 실행이 필요합니다. 네이버 개발자센터에서 「지역 검색」 API가 활성화되어 있는지도 확인해 주세요.

    개인정보·위치 정보는 어떻게 처리되나요?

    「내 위치」 사용 시 좌표는 검색 요청 시 서버(또는 API)로 전달되어 거리 계산에만 쓰이며, 별도 회원 DB에 저장하지 않습니다. 댓글·닉네임은 사용자 기기에만 보관됩니다.
